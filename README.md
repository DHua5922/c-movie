# CMovie

Website for searching movies.

## Technologies

- React
- Vite
- HTML
- CSS
- TypeScript

## How To Run In Development Mode

Enter below:

```bash
yarn dev
```

## Best Practices

- Follow [BEM Naming](https://getbem.com/naming/) conventions for CSS classes.
- Do not use `any` unless there is a very good reason to. `any` defeats the purpose of TypeScript.
